# README النهائي - مساعد لائحة كلية الذكاء الاصطناعي

## 1) نظرة عامة
هذا المشروع عبارة عن مساعد ذكي للإجابة على أسئلة لائحة كلية الذكاء الاصطناعي (جامعة كفر الشيخ) باستخدام:
- Retrieval محلي عبر FAISS + TF-IDF
- توليد إجابة عبر Ollama (محلي)
- واجهة ويب Flask محسنة بصريًا

الهدف: إجابات عربية دقيقة وسريعة، مع مصادر مسترجعة من بيانات اللائحة.

---

## 2) حالة المشروع الحالية
المشروع يعمل محليًا على Windows بنجاح، ويتضمن:
- واجهة Flask حديثة مع تصميم RTL
- تحسينات سرعة (Caching + تقليل حمل التوليد)
- تنسيق إجابة ومصادر بشكل منظم
- زر مسح السابق
- أسئلة سريعة جاهزة

## 5) طريقة التشغيل (النسخة النهائية)
### 5.1 تفعيل البيئة الافتراضية
```powershell
& .\.venv\Scripts\Activate.ps1
```

### 5.2 تثبيت المتطلبات
```powershell
d:/Downloads/files/.venv/Scripts/python.exe -m pip install -r requirements.txt
```

### 5.3 بناء الفهرس من أحدث بيانات
```powershell
d:/Downloads/files/.venv/Scripts/python.exe build_clean_index.py
```

### 5.4 تشغيل Ollama (لو مش شغال)
```powershell
ollama serve
```

### 5.5 تشغيل الواجهة النهائية (Flask)
```powershell
d:/Downloads/files/.venv/Scripts/python.exe flask_app.py
```

افتح المتصفح على:
- http://127.0.0.1:5000

## 6) طريقة العمل داخليًا
1. المستخدم يكتب سؤال في الواجهة.
2. النظام يعمل retrieval من الفهرس المحلي.
3. لو Ollama متاح:
   - يتم توليد إجابة عربية قصيرة من المقاطع المسترجعة.
4. لو Ollama غير متاح:
   - النظام يرجع إجابة retrieval مباشرة.
5. تعرض الإجابة مع:
   - ملخص سريع
   - تنسيق منظم (فقرات/قوائم)
   - مصادر مسترجعة منسقة

## 7) تحسينات الأداء المطبقة
تم تنفيذ تحسينات لتقليل زمن الرد:
- Cache لحالة Ollama (TTL قصير)
- Cache لإجابات الأسئلة المتكررة
- تقليل حجم السياق المرسل للموديل
- تقليل عدد التوكينات المتوقعة في التوليد
- تقليل timeout الخاص بالتوليد

النتيجة العملية: الاستفسارات المتكررة أسرع بشكل واضح.

## 10) استكشاف الأخطاء
### الواجهة لا تفتح
- تأكد أن Flask شغال على 5000
- جرّب إعادة تشغيل:
```powershell
d:/Downloads/files/.venv/Scripts/python.exe flask_app.py
```

### الرد بطيء
- تأكد أن Ollama يعمل محليًا
- جرّب نفس السؤال مرة ثانية (سيستفيد من الكاش)

### لا توجد إجابة مناسبة
- أعد صياغة السؤال بشكل أكثر تحديدًا
- أضف كلمات من نص اللائحة مباشرة

### بعد تعديل data.json النتائج قديمة
- أعد بناء الفهرس:
```powershell
d:/Downloads/files/.venv/Scripts/python.exe build_clean_index.py
```
